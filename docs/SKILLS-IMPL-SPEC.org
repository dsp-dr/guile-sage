* 3. Module skeleton

#+begin_src scheme
;;; skills.scm --- SKILL.md filesystem loader -*- coding: utf-8 -*-

;;; Commentary:
;;
;; Reads SKILL.md files from multiple scope directories, parses YAML
;; frontmatter, and registers each skill as a slash command in the REPL.
;;
;; Contract: docs/SKILLS-CONTRACT.org (invariants SK01-SK24).
;; Companion: docs/SKILLS-IMPL-SPEC.org.
;;
;; Scopes (precedence order, lowest number wins on name collision):
;;   0=enterprise  1=personal  2=project  3=plugin  4=universal  5=claude-compat
;;
;; Public API:
;;   (skills-scan!)                ; scan dirs, register commands
;;   (skills-reload!)              ; re-scan from disk
;;   (skills-reinit!)              ; hard reset for --hard reload
;;   (skills-list)                 ; user-visible skill listing
;;   (skill-show name)             ; full skill alist by name
;;   (skill-invoke name args)      ; invoke skill, inject into session
;;   (skills-paths)                ; resolved scan paths
;;   (skills-scope-set! scope on?) ; stub: scope toggling
;;   (skills-system-prompt-listing) ; description listing for LLM context
;;
;;; reload-contract: destroys *skills*, *skills-scan-paths*,
;;;   *registered-skill-command-keys*; --hard must re-run skills-reinit!.

(define-module (sage skills)
  #:use-module (sage config)
  #:use-module (sage util)
  #:use-module (sage logging)
  #:use-module (sage tools)
  #:use-module (sage hooks)
  #:use-module (sage session)
  #:use-module (srfi srfi-1)
  #:use-module (ice-9 regex)
  #:use-module (ice-9 ftw)
  #:use-module (ice-9 textual-ports)
  #:use-module (ice-9 format)
  #:export (skills-scan!
            skills-reload!
            skills-reinit!
            skills-list
            skill-show
            skill-invoke
            skills-paths
            skills-scope-set!
            skills-system-prompt-listing
            *skills*
            *skills-scan-paths*
            *skills-char-budget*))
#+end_src

* 4. Module-level state

#+begin_src scheme
(define *skills* '())
;; Alist of (name . skill-alist). Highest-precedence skill wins on
;; name collision (enterprise=0 beats personal=1, etc).

(define *skills-scan-paths* '())
;; Ordered list of (scope-sym . absolute-path) pairs resolved at scan time.

(define *skills-char-budget* 8000)
;; Listing budget (SK24). 1% of context window × 4 chars/token,
;; floored at 8000. Override: SAGE_SKILL_TOOL_CHAR_BUDGET env var.

(define *registered-skill-command-keys* '())
;; Tracks which "/name" keys in *commands* (sage repl) were added by
;; this module, so deregister can remove exactly those.
#+end_src

* 5. Skill record structure

Each parsed skill is a plain alist (no SRFI-9 records — matches
=*hooks*= and =*custom-commands*= conventions).

| Key | Type | Default | Notes |
|-----|------|---------|-------|
| ="name"= | string | dirname | SK03: =[a-z0-9-]{1,64}= |
| ="description"= | string | =""= | Recommended. Truncated at 1536 in listing (SK05) |
| ="when_to_use"= | string or =#f= | =#f= | Appended to description in listing |
| ="argument-hint"= | string or =#f= | =#f= | Shown in autocomplete |
| ="user-invocable"= | boolean | =#t= | If =#f=, hidden from =/= menu |
| ="disable-model-invocation"= | boolean | =#f= | If =#t=, model never auto-invokes |
| ="allowed-tools"= | list of string | ='()= | Pre-approve tools for this turn |
| ="model"= | string or =#f= | =#f= | Override active model |
| ="effort"= | string or =#f= | =#f= | =low\vert{}medium\vert{}high\vert{}xhigh\vert{}max= |
| ="context"= | string or =#f= | =#f= | ="fork"= to run in subagent |
| ="agent"= | string or =#f= | =#f= | Subagent type for =context: fork= |
| ="hooks"= | alist or =#f= | =#f= | Skill-scoped lifecycle hooks |
| ="paths"= | list of string | ='()= | Glob patterns gating auto-invocation |
| ="shell"= | string | ="bash"= | For preprocessing |
| ="_scope"= | symbol | — | ='enterprise=, ='personal=, ='project=, etc. |
| ="_scope-priority"= | integer | — | 0-5 |
| ="_skill-dir"= | string | — | Absolute path to skill directory |
| ="_body"= | string | — | Verbatim body (NOT substituted) |
| ="_extra-fields"= | alist | ='()= | SK08: unknown keys preserved |

* 6. Internal helpers

** 6.1 Path resolution

=resolve-scope-paths= → list of =(scope-sym . path)= pairs.

| Priority | Scope | Path | Env override |
|----------|-------|------|-------------|
| 0 | enterprise | =/etc/sage/skills/= | =SAGE_ENTERPRISE_SKILLS_DIR= |
| 1 | personal | =(string-append (xdg-config-home) "/sage/skills/")= | =SAGE_PERSONAL_SKILLS_DIR= |
| 2 | project | =.sage/skills/= relative to =(workspace)= | =SAGE_PROJECT_SKILLS_DIR= |
| 3 | plugin | stub: ='()= in v1 | — |
| 4 | universal | =~/.agents/skills/= | — |
| 5 | claude-compat | =~/.claude/skills/= (only when =SAGE_CLAUDE_COMPAT=1=) | — |

Reuse: =xdg-config-home= from =config.scm:173=, =workspace= from =tools.scm=.

For project scope (§2.3 monorepo): walk upward from =(getcwd)= via
=(dirname)= until hitting =(workspace)= or filesystem root, collecting
any =.sage/skills/= found. Closest ancestor wins on name collision.

** 6.2 YAML frontmatter parser

Two functions:

*** =parse-skill-md= : string → (values alist string)

Splits content on first =---\n...\n---\n= block. Returns
=(values frontmatter-alist body-string)=. If no frontmatter found,
returns =(values '() content)= (SK02). Strips UTF-8 BOM (SK01).

*** =parse-yaml-flat= : string → alist

Line-by-line state machine with three states: =outside=, =in-array=,
=in-hooks=. Handles:

1. =key: value= → string scalar (unquoted or single/double quoted)
2. =key: true|false= → Scheme boolean
3. =key: [item1, item2]= → inline array
4. Multi-line arrays: =key:\n  - item\n  - item=
5. =hooks:= key: depth-1 sub-alist of =event: handler= pairs
6. Unknown nesting (depth >1 on non-hooks keys) → skip with WARN (SK09)

About 60 lines. No regex dependency beyond what =tools.scm= already imports.

*** =parse-yaml-scalar= : string → string | boolean

Strip quotes, handle =true=/=false=/=null=.

** 6.3 Validation

| Function | Behavior |
|----------|----------|
| =validate-skill-name= | Match =[a-z0-9-]{1,64}= via =(string-match)=. Returns string or =#f= (SK03) |
| =normalize-allowed-tools= | Accept string (space-split) or list → list of strings (SK06) |
| =truncate-listing-text= | Combine description + when_to_use, truncate at 1536 chars preserving first clause (SK05, SK23) |
| =symlink-escapes-scope?= | =(string-prefix? scope-root (canonicalize-path-safe path))= — reuse =canonicalize-path-safe= from =tools.scm:133= (SK12) |

** 6.4 Directory scanner

=scan-scope-dir= : scope-sym × priority × path → list of skill-alists

For each subdirectory of =path=:
1. Check =SKILL.md= exists
2. Check file size ≤ 512 KB (SK10)
3. Check symlink does not escape scope root (SK12)
4. Read UTF-8 content
5. =parse-skill-md= → (frontmatter, body)
6. =parse-yaml-flat= inside =catch #t=; on error: =log-warn "skills"= and continue (SK09, SK22)
7. Validate =name= (SK03); on failure: WARN and skip
8. Build skill alist with all fields + =_scope=, =_scope-priority=, =_skill-dir=, =_body=

Uses =(scandir dir-path)= from =(ice-9 ftw)=. No recursion.

=merge-skill-registries= : list of skill-lists → alist of =(name . skill-alist)=

Fold all lists into one alist. Same name: lowest =_scope-priority= wins
(SK13). Same scope: lexicographic tiebreak (SK04). WARN once per
duplicate.

** 6.5 Command registration

=register-skill-command!= : skill-alist → boolean

Mutates =*commands*= in =(sage repl)= via =module-ref= / =module-set!=
(avoids circular import). Pattern:

#+begin_src scheme
(let* ((name (assoc-ref skill "name"))
       (cmd-key (string-append "/" name))
       (repl-mod (resolve-module '(sage repl)))
       (cmds (module-ref repl-mod '*commands*)))
  (if (assoc cmd-key cmds)
      ;; SK: built-in wins, WARN
      (begin
        (log-warn "skills"
                  (format #f "/~a conflicts with built-in; keeping built-in" name)
                  `(("skill-dir" . ,(assoc-ref skill "_skill-dir"))))
        #f)
      (begin
        (module-set! repl-mod '*commands*
                     (cons (cons cmd-key (make-skill-handler skill)) cmds))
        (set! *registered-skill-command-keys*
              (cons cmd-key *registered-skill-command-keys*))
        #t)))
#+end_src

=deregister-skill-commands!= : removes all keys in
=*registered-skill-command-keys*= from =*commands*=. Called by
=skills-reload!= before re-scanning.

=make-skill-handler= : skill-alist → =(lambda (args) ...)=

Closure that calls =invoke-skill=. Signature matches the handler
convention in =repl.scm:1059= (=handle-command= calls =(handler args)=
where =args= is a string).

** 6.6 Invocation pipeline

=invoke-skill= : skill-alist × args-string → string or =#f=

Steps (contract §4.2):
1. Read =_body= from skill alist (stored at scan time)
2. =preprocess-shell-commands body skill= → preprocessed
3. =substitute-variables preprocessed args-string skill= → rendered
4. Log overrides (=allowed-tools=, =model=, =effort=) — stubs in v1 that WARN
5. =session-add-message "user" rendered= (from =session.scm:21=)
6. Register/unregister skill-scoped hooks around invocation
7. Return rendered body

** 6.7 Shell preprocessing (MVP: inline form only)

=preprocess-shell-commands= : body × skill → string

Match =!`([^`]+)`= in body. For each match:

- If =SAGE_DISABLE_SKILL_SHELL=1=: replace with =[shell command execution disabled by policy]=
- Otherwise: run command, replace with stdout

=run-skill-shell-command= : command × skill → string

Uses =primitive-fork + execlp "/bin/sh" "-c" command= pattern from
=capture-argv-in-dir= (=tools.scm:171-197=). Key constraints:
- Working directory: =(assoc-ref skill "_skill-dir")=
- 10-second timeout (SK19) via =waitpid= loop with =alarm=
- On timeout: return =[skill preprocessing timed out]=
- Each output capped at 64 KB
- Total substituted body capped at 256 KB
- NEVER use =(system (format ...))= — see =tools.scm= commentary at line 142

** 6.8 String substitution

=substitute-variables= : body × args-string × skill → string

Substitution order (avoid double-substitution):

| Placeholder | Expansion |
|-------------|-----------|
| =$ARGUMENTS= | full args string |
| =$ARGUMENTS[N]= / =$N= | Nth whitespace-token (0-based) |
| =${SAGE_SESSION_ID}= | =(session-current-name)= or ="unknown"= |
| =${SAGE_SKILL_DIR}= | =(assoc-ref skill "_skill-dir")= |
| =${SAGE_WORKSPACE}= | =(workspace)= |
| =${CLAUDE_SESSION_ID}= | alias when =SAGE_CLAUDE_COMPAT=1= |
| =${CLAUDE_SKILL_DIR}= | alias when =SAGE_CLAUDE_COMPAT=1= |

If =args-string= is non-empty and body does not contain =$ARGUMENTS=:
append ="\n\nARGUMENTS: "= + args-string (§4.4 compat).

Reuse: =string-replace-substring= from =util.scm:30=.

** 6.9 System-prompt listing

=skills-system-prompt-listing= → string

Iterate =*skills*=. Include skills where =disable-model-invocation= is
=#f= (per table in contract §4.2). Format:

#+begin_example
/name — description (truncated at 1536)
#+end_example

Accumulate until =*skills-char-budget*= is exhausted. If truncated,
append =[... N more skills not shown]=.

** 6.10 Skill-scoped hooks

=register-skill-hooks!= / =unregister-skill-hooks!= : skill-alist → void

If =hooks= frontmatter is present, register each event/handler pair
via =hook-register= (=hooks.scm:53=) with name ="skill:<skill-name>:<event>"=.
Unregister after the invocation turn (per contract §11).

In v1, if handler names don't resolve to known procedures: WARN and skip.

** 6.11 Subagent fork (context: fork)

=fork-skill-subagent= : skill × rendered-body → string

Per contract §9 "shortest path": =primitive-fork + execlp "guile3"=
with =-L src=, pipe rendered prompt on stdin, capture stdout.
Synchronous blocking subprocess. Mark in commentary that a full
Agent-tool-style subagent is out of scope.

** 6.12 Character budget

=compute-skills-char-budget= → integer

#+begin_src scheme
(or (and=> (getenv "SAGE_SKILL_TOOL_CHAR_BUDGET") string->number)
    (max 8000 (inexact->exact (round (* (get-token-limit) 0.01 4)))))
#+end_src

  ;; See docs/SKILLS-CONTRACT.org.
  (let ((n (catch #t
             (lambda () (skills-scan!))
             (lambda (key . rest)
               (log-warn "repl" "skills-scan! failed"
                         `(("error" . ,(format #f "~a" key))))
               0))))

      (log-info "repl" (format #f "Loaded ~a skill~a" n (if (= n 1) "" "s")))))

  (let ((listing (skills-system-prompt-listing)))
    (unless (string-null? listing)
      (session-add-message "system"
                           (string-append
                            "=== Available Skills ===\n"
                            listing))))
#+end_src
