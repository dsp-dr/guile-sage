* Safe Tools

The following tools are safe and can be executed without additional permissions:

- ~read_file~ - Read contents of a file within workspace
- ~list_files~ - List files in a directory within workspace
- ~write_file~ - Write content to a file within workspace
- ~edit_file~ - Edit a file by replacing text (search and replace)
- ~glob_files~ - Find files matching glob pattern
- ~search_files~ - Search for pattern in files
- ~git_status~ - Get git status of workspace
- ~git_diff~ - Get git diff of workspace
- ~git_log~ - Get git log of workspace
- ~git_commit~ - Stage files and create a git commit
- ~git_add_note~ - Add a git note to the current HEAD commit

All file operations are restricted to the workspace directory.
