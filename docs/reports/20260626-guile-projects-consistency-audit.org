* Recurring problems (the consistency drift)

1. *guile vs guile3.* Hardcoded =guile=/=guild= + =#!/usr/bin/env guile=
   shebangs that don't exist on a guile3-only host → nothing runs.
   Hit: monkey-interpreter, mas, par (non-FreeBSD branch). Done right
   (auto-detect with =>=3= gate): nimbus, hegel-guile, the-system.
2. *Masked failures — two kinds, pervasive:*
   - shell: =\|\| echo/true=, loops with no exit tracking, no =pipefail= →
     green on crashes (git-scratch SEGFAULT, par, orchestrator, iteral, nimbus,
     the-system compile).
   - *SRFI-64 default runner exits 0 even on failing =test-assert=* — only
     crashes propagate; real regressions show green (hegel-guile, moltbook;
     git-scratch). Fix pattern (from walsh-research): a custom runner that
     =(exit 1)= when fail-count > 0.
3. *Broken help/check/test from a non-POSIX grep pattern.* =?= / =*?= lazy
   quantifier or a =\= line-continuation inside a quoted Scheme string →
   FreeBSD =ugrep= rejects it; masked by an un-=pipefail='d pipe so =gmake=
   still exits 0 (meaning-theory, replay).
4. *=default= goal hides a broken build.* =all: help= (or =check=) means a bare
   =gmake= is green while building/validating nothing (par, meaning-theory,
   replay; the-system default=check).
5. *Actually-broken builds.* llm-scratch (syntax error), nimbus (no =-L= in
   compile), par (missing =(json)=), orchestrator (test path bug).
6. *Missing convention targets.* No =help=: mas, iteral, the-system, gwm,
   walsh-research. No =check= (only =test=): git-scratch, par, walsh-research,
   meaning-theory. No Makefile: gwm.
7. *Sandbox violation.* orchestrator =install= hardcodes =/usr/local= (the
   global CLAUDE.md mandates stay-within-$HOME).

* Recommended guile-project Makefile standard (derived from guile-sage + the good runners)

1. =GUILE ?= $(shell command -v guile3 || command -v guile)= with a =>=3= gate;
   same for =GUILD=. No bare =guile= / =#!/usr/bin/env guile= shebangs.
2. =help= as the *self-documenting default*, with a POSIX-safe pattern
   (no =?=/=*?=/=\= inside the grep; prefer pure =awk=). =build= for artifacts.
3. =check= target (the convention) — alias to =test= if needed, so
   =gmake check= always works.
4. *No masked failures*: custom SRFI-64 runner that =(exit 1)= on any failure
   (like guile-walsh-research); no =\|\| true/echo=; use =set -o pipefail= or
   explicit exit tracking in recipe loops.
5. compile rule carries =-L src= (or =.=) and orders inter-module deps.
6. =install= stays within =$HOME= (XDG paths), never =/usr/local=.
