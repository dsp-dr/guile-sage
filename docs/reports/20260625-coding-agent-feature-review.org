*Platform finding:* 2 of 4 don't run on FreeBSD.
- =opencode-ai= is platform-gated in =package.json= =os= to =darwin,linux,win32=
  → =npm i -g= fails =EBADPLATFORM= on freebsd-x64.
- =droid= (Factory's real npm pkg — verified: homepage Factory-AI/factory,
  @factory.ai maintainers) installs, but the binary aborts:
  =Error: Unsupported platform: freebsd-x64=.
- Implication: OpenCode/Droid must be reviewed from a *Linux/macOS host* (or a
  Linux jail/VM); Claude Code + Codex can be reviewed natively here. This itself
  is a data point — FreeBSD support is a differentiator guile-sage has and these
  two lack.
