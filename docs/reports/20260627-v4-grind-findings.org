* The signal: cross-port AGREEMENT
- *Value-type identity is universal (→ §15.1).* Typed ports (Rust/Haskell/Scala/
  OCaml/Zig) reach it via sum types; untyped string ports (AWK/Tcl) *manufacture*
  it via an explicit ={TYPE PAYLOAD}= tag; hash-reader langs (Racket/Clojure/
  Erlang/JS) get it free. Both extremes converging proves it's a spec requirement,
  not a typed nicety. Only the Guile *reference alist* collapses null/{}/absent.
  Partner: canonical/ordered encoding (Erlang/Tcl) for no-oracle byte-identity.
- *§3 symlink-safe vs "total" is a real contradiction (→ §15.3).* Java + Elixir(v3)
  + Deno + AWK: real symlink resolution does I/O that can throw. Split into a
  pure-total guard + optional effectful pass.
- *§4 error-code taxonomy + no-oracle scope (→ §15.2).* ~all ports: pin
  -32700/-32600/-32601/-32602/-32603; no-oracle = byte-identical error object,
  same id, name-free, =tools/call= only; unknown *method* differs.
- *§6 compaction split (→ §15.4).* Java/Deno/Erlang/Racket/Rust: =importance/
  summarize/intent= are named but undefined → split pure (testable) vs LLM-backed;
  pin auto-compact at 0.9×limit.
- *§8 spawn → generic native process API (→ §15.5).* Babashka/Elisp/Racket/Rust:
  no SIGSEGV class — the Guile rule is impl-specific. Backoff base 200ms·2^n
  (Java/Deno/Zig/Rust).
- *§2 encode-first + label buckets (→ §15.6).* Tcl/Io/Elisp/Erlang/Zig: "≤200
  bytes UTF-8-safe" assumes byte strings; decoded-string runtimes must encode
  first; sanitize must never *introduce* invalidity; =<label>= is illustrative.
- *§7 correct CDATA invariant (→ §15.7).* Zig + Deno + Rust: round-trip identity +
  =]]>=-count, NOT cdata-nesting/balance. §4/§7 served-plain cross-ref (Tcl).
