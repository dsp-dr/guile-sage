* Project Status: v0.6.0

** Summary

guile-sage is a functional AI REPL written in Guile Scheme that provides:
- Interactive chat with Ollama-compatible LLMs
- Tool calling system (22 tools)
- Session management with persistence
- Self-modification capabilities
- Context compaction for long conversations

*** Tool Categories

**** Safe Tools (18 total)
- File: read_file, list_files, write_file, edit_file, glob_files, search_files
- Git: git_status, git_diff, git_log, git_commit, git_add_note, git_push
- Logs: read_logs, search_logs
- Tasks: sage_task_create, sage_task_complete, sage_task_list, sage_task_status

**** Unsafe Tools (4 total, require YOLO mode)
- run_tests (shell execution)
- eval_scheme (arbitrary code)
- reload_module (module system)
- create_tool (dynamic registration)

** Configuration

*** Required
- Guile 3.0+
- curl
- Ollama (local or cloud)

*** Environment Variables

| Variable | Default | Description |
|----------+---------+-------------|
| SAGE_OLLAMA_HOST | http://localhost:11434 | Ollama API endpoint |
| SAGE_MODEL | (provider default) | Model to use |
| OLLAMA_API_KEY | (none) | API key for cloud |
| SAGE_YOLO_MODE | (unset) | Enable unsafe tools |
| SAGE_LOG_LEVEL | info | Log level |
